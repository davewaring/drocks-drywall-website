# D-Rocks Dynamic Drywall Repair Website

## Project Overview
Single-page website for D-Rocks Dynamic Drywall Repair LLC, a drywall and handyman company serving Sussex County, NJ.

**Live Domain:** drocksllc.com (to be configured on Vercel)
**GitHub:** https://github.com/davewaring/drocks-drywall-website

## Tech Stack
- Next.js 15 with React 19
- TypeScript
- Tailwind CSS 4
- Static export for Vercel deployment
